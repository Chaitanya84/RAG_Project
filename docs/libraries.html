🌐 Streamlit

Install: pip install streamlit · Import: import streamlit as st · Used in: all page files

Streamlit turns Python scripts into interactive web apps. You write Python, it renders HTML. Every time a user clicks a button or types text, Streamlit re-runs your entire script from top to bottom.

import streamlit as st

st.title("Hello World")
name = st.text_input("What's your name?")
if name:
    st.write(f"Hello, {name}!")

Key widgets used in this project:

  • st.file_uploader() — Upload PDF files
  • st.chat_input() / st.chat_message() — Chat interface
  • st.form() / st.radio() — Quiz form with radio buttons
  • st.session_state — Persistent dictionary across page re-runs
  • @st.cache_resource — Cache expensive computations (like loading embeddings)
  • st.spinner() — Loading animation

🤖 OpenAI

Install: pip install openai · Import: from openai import OpenAI · Used in: AnswerAll.py, embedandchunk.py

The official Python client for OpenAI's API. We use two endpoints:

  1. Embeddings API — Converts text into numerical vectors (1,536 numbers).
  2. Chat Completions API — Sends a prompt and gets a text response (like ChatGPT).
from openai import OpenAI

client = OpenAI(api_key="sk-proj-...")

# Embeddings
response = client.embeddings.create(
    model="text-embedding-3-small",
    input=["Hello world"],
)
vector = response.data[0].embedding  # List of 1,536 floats

# Chat
response = client.chat.completions.create(
    model="gpt-4o-mini",
    messages=[{"role": "user", "content": "What is RAG?"}],
)
answer = response.choices[0].message.content

📄 PyMuPDF (fitz)

Install: pip install PyMuPDF · Import: import fitz · Used in: embedandchunk.py

Reads PDF files and extracts text from each page. The import name fitz comes from the underlying C library (MuPDF) by Artifex. It's fast and handles most PDF formats including scanned text (if OCR was applied).

import fitz

doc = fitz.open("document.pdf")
for page in doc:
    text = page.get_text()   # Plain text of one page
    print(text[:100])
doc.close()
💡 Why not pdfplumber or PyPDF2? PyMuPDF is significantly faster (10–50x) and uses less memory. It also handles edge cases like embedded fonts and rotated pages better.

📊 pandas

Install: pip install pandas · Import: import pandas as pd · Used in: embedandchunk.py, AnswerAll.py

The standard Python library for working with tabular data (rows and columns). We use it to read/write CSV files and convert them to lists of dictionaries.

import pandas as pd

# Read a CSV file
df = pd.read_csv("chunks.csv")
print(df.shape)        # (1366, 6) → 1,366 rows, 6 columns
print(df.columns)      # ['source_file', 'page_number', 'sentence_chunk', ...]

# Convert to list of dicts (used by AnswerAll.py)
records = df.to_dict(orient="records")
# [{"source_file": "book.pdf", "page_number": 1, ...}, ...]

# Write a DataFrame to CSV
df.to_csv("output.csv", index=False)

🔢 NumPy

Install: pip install numpy · Import: import numpy as np · Used in: AnswerAll.py, embedandchunk.py

NumPy provides fast array operations written in C. Pure Python loops over 1,366 vectors × 1,536 dimensions would take seconds. NumPy does it in milliseconds.

import numpy as np

# Create an array
a = np.array([1.0, 2.0, 3.0])

# Vector norm (length)
print(np.linalg.norm(a))  # 3.7416...

# Matrix multiplication (used for cosine similarity)
b = np.array([[1, 0, 0], [0, 1, 0]])
print(b @ a)  # [1., 2.] — dot product of each row with a

# Sorting indices (used to find top-k chunks)
scores = np.array([0.3, 0.9, 0.1, 0.7])
top_3 = np.argsort(scores)[::-1][:3]  # [1, 3, 0] — indices of highest scores

⏳ tqdm

Install: pip install tqdm · Import: from tqdm.auto import tqdm · Used in: embedandchunk.py

Wraps any loop or iterable with a smart progress bar. Used during PDF processing and embedding creation so you can see how far along the operation is.

from tqdm.auto import tqdm

for item in tqdm(range(100), desc="Processing"):
    do_work(item)

# Output: Processing: 45%|████████▌          | 45/100 [00:02<00:03]

🔄 tenacity

Install: pip install tenacity · Import: from tenacity import retry, ... · Used in: AnswerAll.py, embedandchunk.py

Automatically retries function calls that fail, with configurable delays. Essential for API calls that can fail due to rate limits, timeouts, or transient network issues.

from tenacity import retry, stop_after_attempt, wait_exponential

@retry(
    stop=stop_after_attempt(4),                         # Give up after 4 tries
    wait=wait_exponential(multiplier=1, min=2, max=30), # Wait 2s, 4s, 8s...
)
def call_api():
    # If this raises an exception, tenacity retries automatically
    return requests.get("https://api.example.com/data")

Without tenacity, a single API timeout would crash the entire processing pipeline. With it, the app gracefully handles temporary failures.

🔐 python-dotenv

Install: pip install python-dotenv · Import: from dotenv import load_dotenv · Used in: config.py

Reads key-value pairs from a .env file and sets them as environment variables. This is how we keep secrets (API keys) out of source code.

# .env file:
# OPENAI_API_KEY=sk-proj-abc123
# DEBUG=true

from dotenv import load_dotenv
import os

load_dotenv()  # Reads .env file
key = os.getenv("OPENAI_API_KEY")  # "sk-proj-abc123"

🔑 google-auth & google-auth-oauthlib

Install: pip install google-auth google-auth-oauthlib · Used in: app.py

Two libraries that work together for Google OAuth login:

  • google-auth-oauthlib — Builds the OAuth2 login flow (creating the login URL, handling the callback).
  • google-auth — Verifies the ID token Google returns after login, extracting the user's email.
from google.oauth2 import id_token
from google.auth.transport import requests as google_requests

# Verify the token Google sent back
id_info = id_token.verify_oauth2_token(
    token,
    google_requests.Request(),
    client_id,
)
email = id_info["email"]  # "user@gmail.com"

🌍 requests

Install: pip install requests · Import: import requests · Used in: app.py

The most popular Python HTTP client. Used in this project to exchange the OAuth authorization code for tokens by making a POST request to Google's token endpoint.

import requests

# Exchange auth code for tokens
response = requests.post("https://oauth2.googleapis.com/token", data={
    "code": auth_code,
    "client_id": client_id,
    "client_secret": client_secret,
    "redirect_uri": redirect_uri,
    "grant_type": "authorization_code",
})
tokens = response.json()  # {"id_token": "...", "access_token": "..."}
