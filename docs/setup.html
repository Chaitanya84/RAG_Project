⚙️ 2. Setup & Installation

Everything you need to run the app on your own computer. Follow these steps in order.

Prerequisites

  • Python 3.10+ (we use 3.12) — Download
  • Git — to clone the repository
  • An OpenAI API key — requires an OpenAI account with billing enabled
  • Google Cloud credentials — for the OAuth login (optional for local testing)

1 Clone the Repository

git clone https://github.com/YOUR_USERNAME/RAG_Project.git
cd RAG_Project/RAG_Streamlit_App
💡 What is git clone? It downloads a copy of the entire project (code + history) from GitHub to your computer. You only need to do this once.

2 Create a Virtual Environment

# Create the virtual environment
python -m venv .venv

# Activate it
# On Linux/Mac:
source .venv/bin/activate

# On Windows:
.venv\Scripts\activate
💡 What is a Virtual Environment? It's an isolated folder where Python packages are installed. Without it, packages install globally and can conflict with other projects. With it, each project has its own clean set of packages. The .venv folder contains the isolated Python + packages.

You'll know it's active when your terminal prompt shows (.venv) at the beginning.

3 Install Dependencies

pip install -r requirements.txt

This reads the requirements.txt file and installs all listed packages:

streamlit>=1.30.0
openai>=1.0.0
PyMuPDF>=1.23.0
pandas>=2.0.0
numpy>=1.24.0
tqdm>=4.65.0
tenacity>=8.1.0
python-dotenv>=1.0.0
google-auth-oauthlib
google-auth
requests
💡 What does >=1.30.0 mean? "Install version 1.30.0 or any newer version." This ensures you get bug fixes and improvements while maintaining compatibility. If no version is specified (like requests), pip installs the latest available version.

4 Get an OpenAI API Key

  1. Go to platform.openai.com/api-keys
  2. Sign up or log in
  3. Click "Create new secret key"
  4. Copy the key (starts with sk-proj-) — you can only see it once!
  5. Add billing info under Settings → Billing (the free tier won't work for embeddings)
💰 Cost estimate: Processing a 100-page PDF costs roughly $0.01–0.05 (embeddings). Each question/answer costs about $0.001 (chat). A full day of testing might cost $0.50–$2.00 total.

5 Create a .env File

Copy the example and fill in your values:

cp .env.example .env

Then edit .env:

OPENAI_API_KEY=sk-proj-your-key-here
OPENAI_EMBEDDING_MODEL=text-embedding-3-small
OPENAI_CHAT_MODEL=gpt-4o-mini
DEBUG=true
REDIRECT_URI=http://localhost:8501/
⚠️ Never commit .env to Git! It contains your secret API key. The .gitignore file already excludes it. If you accidentally commit it, rotate your key immediately on the OpenAI dashboard.

6 Google OAuth Setup (Optional)

The login page uses Google OAuth. To enable it:

  1. Go to Google Cloud Console
  2. Create a new project (or select an existing one)
  3. Navigate to APIs & Services → Credentials
  4. Click "Create Credentials" → "OAuth 2.0 Client ID"
  5. Set application type to "Web application"
  6. Add http://localhost:8501/ as an Authorized redirect URI
  7. Download the JSON file and save it as client_secret.json in the RAG_Streamlit_App/ folder
💡 What is OAuth? A protocol that lets users log in via Google (or GitHub, Facebook, etc.) without creating a new account on your app. Your app never sees the user's Google password — it only gets their email address after they approve access on Google's own login page.

7 Run the App

streamlit run app.py

Streamlit will print:

  You can now view your Streamlit app in your browser.
  Local URL: http://localhost:8501

Open that URL in your browser. You should see the login page!

Troubleshooting

ProblemSolution
ModuleNotFoundErrorMake sure your virtual environment is activated (source .venv/bin/activate)
OPENAI_API_KEY is not setCheck that .env exists and the key is correct
Port 8501 already in usestreamlit run app.py --server.port=8502
OAuth redirect failsEnsure REDIRECT_URI in .env matches the Google Console setting exactly
