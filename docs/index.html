📁 Project File Structure

RAG_Project/
├── README.md
├── RAG_Streamlit_App/
│   ├── app.py                  ← Entry point (Google OAuth login)
│   ├── config.py               ← All settings, paths, API keys
│   ├── embedandchunk.py        ← PDF → sentences → chunks → embeddings
│   ├── AnswerAll.py            ← Search, answer, and quiz generation
│   ├── requirements.txt        ← Python dependencies
│   ├── Procfile                ← Render start command
│   ├── runtime.txt             ← Python version for Render
│   ├── .env.example            ← Template for environment variables
│   ├── .streamlit/
│   │   └── config.toml         ← Streamlit server settings
│   ├── data/                   ← Generated CSVs, embeddings (gitignored)
│   └── pages/
│       ├── mainPage.py         ← PDF upload & processing
│       ├── chatPage.py         ← RAG chat interface
│       └── quizPage.py         ← Knowledge quiz
